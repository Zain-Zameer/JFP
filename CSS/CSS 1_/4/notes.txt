browser adds random margin and padding so we sometimes need to remove them for efficient designs.

padding is inside the content
margin is outside the content 
because of dpi factor in search engines, actual border dekhne ko nh milta humein lets say 2px ka border dia toh 1.6 something show hoga.

now lets check that how element width got calculated.

Total Width = Width + Left Padding + Right Padding + Left Border + Right Border + Left Margin + Right Margin

Total Height = Height + Top Padding + Bottom Padding + Top Border + Bottom Border + Top Margin + Bottom Margin

You don’t include margin in box-sizing: border-box; calculations because margin is external spacing.

so when we add box-sizing border box it means we are telling that yeh joh height mein dera 200px yeh border or padding ko mila kr batara hoon, margin ko nahi likin

agr mein box-sizing border box nahi likhta toh joh mein height deta woh plus usmein padding lagti plus usmein border lagta toh kitna bara element ka size hojata

Element’s height (box-sizing: border-box) = 200px (includes padding & border).
Total height in layout (including margins) = 210px (adds top & bottom margin).
